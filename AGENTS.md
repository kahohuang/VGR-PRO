# 项目背景

我是一名 TikTok 运营，目前负责 VGR PRO 项目的达人运营工作。我的核心职责是找到美区 TikTok 上适合我们产品的达人，并通过沟通推动他们与品牌达成合作。合作后，达人会在自己的 TikTok 账号发布我们产品的带货视频。

## 公司和项目现状

VGR PRO 当前在 TikTok Shop 美区主要做理发器、修剪器等男士理容产品。项目当前的核心增长模式是：

1. 寻找适合产品的美区达人。
2. 与达人建联、沟通合作细节。
3. 给达人寄样或推动达人申请样品。
4. 达人发布带货视频。
5. 达人提供授权后，公司对优质视频进行投流放大。
6. 通过达人视频和广告放大带动 TikTok Shop 成交。

目前项目不是直播驱动型，而是明显依赖“达人视频 + 广告投流”的内容带货模型。达人视频是核心成交来源，广告投流用于放大已经验证过的优质视频素材。

## 主推产品

当前主推产品是 VGR 886 理发器套装。它是项目目前最重要的单品，也是店铺 GMV 的主要来源。

VGR 886 的核心卖点包括：

1. 理发器 7000-9000 RPM 可调速电机，修剪器 7000-8000 RPM 电机。
2. DLC 刀头，耐用且锋利。
3. 4 小时续航，支持 Type-C 和无线充电。
4. 阿根廷国家足球队官方背书。
5. 电子显示屏，显示电量、转速和缺油提醒。
6. 配备 10 款导梳和可更换保护套握把。
7. 左右渐变导梳是面向家庭用户的重要卖点。
8. 金属机身、人体工学握把、低噪音。
9. 定位为家庭理发器中的高性价比产品，也适合新手理发师入门使用。

除 VGR 886 外，项目中也有 VGR 035、VGR 001 等产品，但现阶段主要成交仍集中在 886。

## 达人合作模式

常规合作方式：

1. 免费给达人提供样品。
2. 要求达人围绕产品卖点和使用场景创作带货视频。
3. 对部分优质视频或优质达人，可以支付一定创意费用。
4. 视频发布后，达人需要同步授权信息，方便公司投流。
5. 公司会对表现好的视频投放 GMV Max 等广告资源。

视频内容重点：

1. 展示真实理发过程，例如渐变、修边、日常剪发。
2. 展示产品实际切割能力、续航、刀头、低噪音、无线充电等卖点。
3. 面向节日节点时，例如父亲节，可以从“父亲节礼物”“在家给父亲理发”“替换旧理发器”等角度创作。
4. 达人收到样品后，通常希望在 2 周内发布产品视频。

## 授权口径

授权状态字段需要区分两种情况：

1. `ad code`：达人给出的单条视频授权。它只授权某一条视频用于广告投放。在表格里，授权状态字段中那些看起来很乱、类似加密字符串的代码，就是达人提供的 `ad code`，不是异常数据。
2. `账号已授权`：达人把整个 TikTok 账号授权给我们。这个权限范围比单条视频的 ad code 更大。

分析数据时不要把 `ad code` 和 `账号已授权` 混为一类。`ad code` 更适合判断单条视频是否可投流，`账号已授权` 更适合判断达人账号是否具备长期投放和素材复用价值。

## 数据文件认知

项目文件夹中的数据大致分为几类：

1. `个人达人建联表.xlsx`：我的个人达人建联工作台，是判断我个人工作量、建联进度、合作漏斗、父亲节专项进展和视频授权状态的关键文件。
2. `VGRPRO每月店铺数据`：店铺 1-4 月整体 GMV、订单、客户、曝光、点击、商品卡 GMV、视频归因 GMV、联盟视频 GMV 等总盘数据。
3. `VGRPRO每月商品详细数据`：各 SKU 的 GMV、订单、曝光、点击、加购、商品状态等数据，用于判断产品结构和主推 SKU。
4. `VGRPRO每月达人视频数据`：达人带货视频的 GMV、订单、曝光、佣金、退款、点赞评论等数据，用于判断哪些达人和视频真正带来成交。
5. `VGRPRO店铺每月投流数据`：广告投流视频的数据，包括总收入、成本、ROI、订单、曝光、点击、授权类型、授权状态等，用于判断视频放量效率。
6. `达人建联话术`：日常建联、父亲节活动、样品申请、发布催促、卖点介绍等沟通话术。
7. `商品链接.txt`：主要 TikTok Shop 商品链接，包括 VGR886、VGR035、VGR001。

## 已形成的项目判断

目前公司处在“单品跑通、达人跑通、广告跑通，正在规模化复制”的阶段。项目已经证明 VGR 886 能通过达人视频成交，也已经证明部分达人视频可以通过投流继续放大。

关键判断：

1. 店铺 GMV 从 2026 年 1 月到 4 月增长明显，真正放量发生在 3 月和 4 月。
2. GMV 主要来自达人视频和联盟视频，不是直播。
3. VGR 886 是绝对主力 SKU，其他 SKU 还没有形成稳定成交。
4. 广告 ROI 整体可接受，优质达人视频具备继续放大的价值。
5. 核心达人包括 `chythebarber`、`gzalez_barbertx`、`millionare.mentaliity`、`jesus_jisai`、`themenbarberschool` 等。
6. 项目后续增长的关键，是持续找到新达人、新素材，并把优质视频转化为可投流资产。

## 我的个人工作情况

我的工作处在项目增长链路的最前端，决定后续是否有足够的新达人、新视频和新广告素材。

从个人达人建联表看，我的主要工作包括：

1. 批量寻找美区 TikTok 达人。
2. 判断达人是否适合 VGR PRO 产品。
3. 发送初步建联话术。
4. 推动达人添加 WhatsApp 或继续在 TikTok 沟通。
5. 推动达人申请样品或接受寄样。
6. 跟进达人是否收到样品。
7. 催促达人按期发布视频。
8. 收集达人发布后提供的 `ad code` 或账号授权。
9. 配合父亲节等营销节点推进专项达人合作。

目前个人工作的重点不是单纯增加建联数量，而是提高从“初步建联/二次建联”到“寄样/达成合作/视频发布/可投流授权”的转化效率。

## 分析项目时的注意事项

后续分析时需要注意：

1. 店铺增长不能只看 GMV，还要同时看退款、取消退货、客单价、曝光、点击和视频归因结构。
2. 商品分析要重点看 VGR 886 是否仍是单品依赖，以及 VGR 035、VGR 001 是否开始跑出新成交。
3. 达人分析要区分自然带货表现和投流后表现。
4. 投流分析要重点看收入、成本、ROI、订单、授权类型和可复用视频数量。
5. 建联分析要重点看漏斗：建联人数、回复人数、WhatsApp 添加、寄样、达成合作、视频发布、ad code、账号授权。
6. 父亲节等活动分析要单独看，因为它是短期营销节点，不能完全等同于日常表现。
7. 授权状态字段里可能同时出现 `ad code`、`账号已授权` 和其他备注。看起来很乱的长代码通常就是 `ad code`，代表单条视频授权；`账号已授权` 代表达人把整个账号授权给我们。
